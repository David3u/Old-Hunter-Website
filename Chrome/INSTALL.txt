INSTRUCTIONS TO INSTALL THE CHROME VERSION OF: Old-Hunter-Website

1. Download chrome_src.zip from the latest release [here.](https://github.com/David3u/Old-Hunter-Website/releases).
2. Unzip the file.
3. Open Chrome and go to chrome://extensions.
4. Enable "Developer mode" (toggle in the top right corner).
5. Click on "Load unpacked" and select the unpacked chrome_src folder.
